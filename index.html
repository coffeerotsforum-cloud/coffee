📤 FileTransfer

Простая передача файлов через браузер
📂
Перетащите файл сюда или нажмите для выбора
💡 Как это работает:
• Файл конвертируется в ссылку прямо в вашем браузере
• Ссылка работает только на этом устройстве
• Идеально для быстрой передачи между устройствами в одной сети
• Файлы НЕ загружаются на сервер - всё локально
